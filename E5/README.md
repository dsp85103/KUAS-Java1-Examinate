# E5 奇偶數差計算
讓使用者輸入一個純數字之字串，並且將其奇位數的數字加總為 A、偶位數的數 字分別加總為 B，並且取 A-B 的絕對值，判斷其是否為 11 的倍數，
公式如下
輸入 3020104010
3+2+1+4+1=11 (A)
0+0+0+0+0=0 (B) 
A-B=11-0=11
11 為 11 的倍數
#
```
> 輸入範例 1： 
> > 請輸入數字：3020104010 
> 輸出範例 1：
> > 3020104010 之奇偶數差為 11 的倍數

輸入範例 2： 
請輸入數字：3020104011 
輸出範例 2：
3020104011 之奇偶數差不是 11 的倍數
```
